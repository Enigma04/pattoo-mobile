Pattoo Mobile
=============

``pattoo`` stores timeseries data in a database and makes it available for users via a GraphQL API. ``pattoo-mobile`` is a mobile application for accessing the data.

Visit the `Pattoo Mobile GitHub site <https://github.com/PalisadoesFoundation/pattoo-mobile>`_ to see the code.
