Acessing Database Information
==============================

Once logged in, all screens shoudl be void of data. 

.. image:: images/PattoAgentsNoData.png
  :width: 300
  
.. image:: images/FavouritesNoData.png
  :width: 300
  
.. image:: images/ChartsNoData.png
  :width: 300

To resolve this issue. The user should navigate to the settings screen. 

The user should see two options, the first to activate dark mode, and the other enter a url which will act as the source to retrive the data from the pattoo database. 

.. image:: images/SettingsScreen.png
  :width: 300
  

To access the data, enter the desired url.


.. image:: images/urlenterednotgreen.png
  :width: 300
  

If the url was entered correclty, there will be an indcator. If indicator not see, re-enter or check if you have entered a valid url. 
  
.. image:: images/urlvalidated.png
  :width: 300
